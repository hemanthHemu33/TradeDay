# TradeDay

TradeDay is a TypeScript-first scaffold for an event-driven trading platform. This repository now includes the requested domain-oriented folder layout, starter docs, and placeholder modules for future implementation.

## Getting started

1. Install dependencies: `npm install`
2. Review `.env` and update local values.
3. Start the dev entrypoint: `npm run dev`

## Repository highlights

- `docs/`: operational and product reference documents.
- `src/`: application modules organized by domain.
- `scripts/`: runnable entrypoints for local workflows.
